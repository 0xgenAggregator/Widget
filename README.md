## 0xGen Widget

The 0xgen widget is a Vue component that can be easily imported into any Vue project.

### Install
```javascript
git clone https://github.com/oxgen/widget.git
cd loading
npm install
```

### Run
```javascript
npm run example
```

### Compile
```javascript
npm run compile
```

### Npm
```javascript
npm install @0xgen/widget
```

### Usage

```javascript
import Vue from 'vue'
import widget from '@0xgen/widget'

Vue.use(widget);
```
