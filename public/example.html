Quick Start

We highly recommend using custom mode to integrate wallets to make sure wallets work properly.
Read more details HERE

Method one - Query Widget.js in your <script> tag.

1. Step

  • 01. input <script> tag in your front-end code.
  • 02. change the script src to "https://app.0xgen.io/widget.js"
  • 03. refresh the page and check whether the component is installed.

2. Code

<script src="https://app.0xgen.io/widget.js"></script>

Method two - Use <iframe> to inject the Gen widget into your program.

1. Step

  • 01. input <iframe> to your front-end code.
  • 02. change the iframe src to "https://app.0xgen.io"
  • 03. refresh the page and check whether the component is installed.

2. Code

<iframe src="https://app.0xgen.io"></iframe>
